# Buttons

Buttons call on users to take an action.
They should always use a verb in the imperative mood so users know what to expect from clicking the button.
Each page should generally have at most one _primary_ call to action &mdash; the main thing you want a user to do.
Primary buttons should be styled to draw the user's eye to the primary call to action without distracting them significantly.

Other, less critical calls to action should use a secondary button styling.
Secondary buttons are more muted in appearance but still provide clear expectations about their function.
